\section{利用等式性质解方程}
\begin{frame}
    \frametitle{利用等式性质解方程}
    \makecenter{利用等式的性质解下列方程}
    \myblock{1-}{}{
        \begin{enumerate}[i]
            \item \bmc{x+7=26}
            \item \bmc{-5x=20}
            \item \bmc{-\dfrac{1}{3}x-5=4}
        \end{enumerate} 
    }
    \only<2>{
        \begin{description}
            \item[解：] 第一个方程两边同时减去$7$, 得：
                \begin{align}
                    \bm{x +7 -7 = 26 -7} \\
                    \bm{x=19}
                \end{align}
        \end{description}
    }
    \only<2>{
        \myblock{2}{}{解一元一次方程要 化归 为 \alert{$x=a$(常数)}的形式.
        }
    }
\end{frame}

\begin{frame}{检验方程的解}
    \makecenter{
        一般地，从方程解出未知数的值以后，可以代入原方程检验，
        看这个值能否使方程的两边相等。
    }
    \myblock{1-}{例如}{
        将$x=-27$ 带入方程 \bmc{-\dfrac{1}{3}x-5=4}的左边，\\
        \bmc{-\dfrac{1}{3}\times (-27)-5 = 9-5 = 4}\\
        \vspace{1em}
        方程左右两边相等，所以 $x=27$是原方程的解。
    }
\end{frame}
